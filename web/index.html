Getting Started

📋 Setup Instructions

  1. Database Setup: Run the SQL files in your Supabase project:
    • 1. dnd_normalized_schema.sql
    • 2. dnd_reference_data.sql
    • 3. dnd_sample_characters_normalized.sql
  2. Configuration: Update the Supabase URL and API key in js/database.js
  3. Characters: Start with the sample characters or create your own
  4. Campaign: Begin your adventure with fully tracked characters!
